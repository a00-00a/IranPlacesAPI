# Iran Places API

A RESTful API that provides information about Iran's provinces and cities. This project is developed using **Node.js**, **Express**, and **TypeScript**.

## Features

- Retrieve a full list of provinces and cities
- Search for a province by Name or ID
- Search for a city by Name or ID
- Get a list of cities within a specific province
- Find the province associated with a specific city
- CORS support for cross-domain access

## Prerequisites

- Node.js (Latest stable version recommended)
- npm
